# TurnSarsah Frontend (Vite + React + TypeScript)

Minimal scaffold for a frontend connected to the TurnSarsah project.

Quick start

```bash
cd frontend
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Notes

- Uses Vite with `@vitejs/plugin-react` and TypeScript.
- Entry: `src/main.tsx` -> `src/App.tsx`.
