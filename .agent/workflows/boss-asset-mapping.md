# 보스 에셋 매핑 작업 원칙

앞으로 챕터 및 스테이지가 추가됨에 따라 보스의 이미지, 공격 효과음, 상태이상 등을 매핑할 때 다음 원칙을 반드시 준수한다.

1. **패턴 기반 매핑**: 기존 챕터별 관리 패턴(파일명 컨벤션, 경로 구조 등)을 분석하여 동일한 방식으로 구현한다.
2. **임의 적용 금지**: 매핑할 이미지나 효과음 파일을 찾을 수 없거나 정의되지 않은 경우, **절대로 임의의 플레이스홀더나 다른 에셋을 적용하지 않는다.**
3. **사용자 확인 필수**: 필요한 에셋이 없거나 불확실한 경우, 반드시 사용자에게 상황을 설명하고 **"에셋이 없으므로 어떻게 진행해야 할까요?"**라고 질문하여 가이드를 받는다.
4. **일관성 유지**: 모든 챕터와 스테이지에서 에셋 관리의 일관성을 유지한다.
