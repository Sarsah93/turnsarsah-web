# 턴살사 (TurnSarsah) - 유저 매뉴얼

턴살사는 포커 족보를 활용한 1:1 턴제 카드 배틀 게임입니다. 플레이어는 카드를 선택하여 강력한 족보를 완성하고, 보스의 체력을 0으로 만들어 스테이지를 클리어해야 합니다.

---

## 1. 게임 기본 규칙
- **목표**: 각 챕터의 스테이지 보스들을 처치하여 최종 승리를 거둡니다.
- **체력(HP)**: 플레이어의 HP가 0이 되면 패배, 보스의 HP가 0이 되면 스테이지 클리어.
- **드로우 및 턴**: 
  - 매 턴 기본 2회의 카드 교체(SWAP) 기회가 주어집니다. 
  - 공격 후 사용한 카드는 자동으로 보충됩니다.

---

## 2. 공격 및 데미지 계산
족보 점수와 카드 숫자의 합산으로 데미지가 결정됩니다.

### 족보 보너스 (Hand Bonus)
| 족보 | 보너스 점수 |
| :--- | :--- |
| **Royal Flush** | 300 |
| **Straight Flush** | 175 |
| **Four of a Kind** | 150 |
| **Full House** | 125 |
| **Flush** | 100 |
| **Straight** | 75 (Wrap-around 포함) |
| **Three of a Kind** | 50 |
| **Two Pair** | 20 |
| **One Pair** | 10 |
| **High Card** | 0 (가장 높은 카드 2장의 합) |

- **조커(Joker)**: 모든 카드를 대체하며, 가장 높은 가치(14점)를 가집니다.
- **크리티컬**: 족보 구성에 A나 조커가 포함될 확률로 데미지가 증폭됩니다.
- **특수 고정 데미지**: 일부 보스전(2A-10)에서는 족보와 무관한 특수 계산식이 적용될 수 있습니다.

---

## 3. 상태 이상 및 중첩 규칙
전투 중 발생하는 변수로, 다양한 효과가 부여될 수 있습니다.

| 효과 | 설명 | 지속/중첩 |
| :--- | :--- | :--- |
| **출혈 (Bleeding)** | 매 턴 5 데미지 부여 | 4~6턴 |
| **과출혈** | 출혈 중첩 시 발생, 15 데미지 부여 | 6턴 |
| **중독 (Poisoning)** | 턴마다 데미지가 5씩 점진적 증가 | 3~4턴 |
| **쇠약 (Debilitating)**| 최대 체력 및 가하는 데미지 20% 감소 | 3턴 |
| **마비 (Paralyzing)** | 한 턴 동안 공격 불가능 (Turn Skip) | 2턴 |
| **신경성 맹독 (Neuro)** | 매 턴 15 데미지 및 **명중률 30% 감소** | 3턴 |
| **면역 (Immune)** | 모든 신규 디버프 차단 및 **부여 즉시 기존 디버프 정화** | 3턴 |
| **재생 (Regen)** | 매 턴 체력을 일정량 회복 | 상시/특수 |
| **경감 (Reduction)** | 받는 데미지를 일정 비율(5~40%) 감소 | 상시 |
| **회피 (Avoiding)** | 일정 확률로 적의 공격을 완전히 무효화 | 상시 |

---

## 4. UI 및 조작 가이드
- **공격 (ATTACK)**: 현재 선택한 카드들로 보스를 공격합니다. (최대 5장)
- **교환 (SWAP)**: 선택한 카드를 덱의 새로운 카드로 바꿉니다.
- **콤보 미리보기 (Combo Preview)**: 카드 선택 시 예상 데미지와 족보 이름을 실시간으로 보여줍니다.
- **상태이상 아이콘**: HP바 근처 아이콘을 클릭하여 상세 설명을 볼 수 있습니다.

---

## 5. 챕터별 보스 데이터 매뉴얼

### 🌳 챕터 1: 들판 (Meadow field)
*   **주요 규칙**: 카드 사용 제약(Rank/Suit Ban), 카드 뒤집기(Blind).
*   **핵심 기믹**: 
    - **공격력 성장**: 보스들은 공격 성공 시마다 공격력이 상승하는 패턴을 가집니다.
    - **피해 경감**: 후반부 보스(8~10)들은 플레이어의 피해를 10~20% 상시 경감합니다.
    - **각성**: 최종 보스 '고블린 로드'는 체력이 절반 이하가 되면 모든 능력을 복구하며 각성합니다.

### 🏜️ 챕터 2A: 사막 (Desert)
*   **주요 규칙**: 특정 족보 무효화(DMG 0), 강제 카드 교체, 수수께끼(Puzzle).
*   **핵심 기믹**: 
    - **환경적 압박**: 챕터 2A의 보스들은 **중독**과 **쇠약** 상태이상을 빈번하게 부여하여 플레이어의 유지력을 시험합니다.
    - **상시 패시브**: 대부분의 보스가 **재생, 피해 경감, 회피** 중 하나 이상의 방어적 패시브를 보유합니다.
    - **BANNED 카드**: 2A-8에서는 특정 슬롯의 카드가 금지되어 점수 기여도가 0이 됩니다.
    - **스 Sphinx의 수수께끼**: 최종 스테이지에서는 주어진 목표 숫자를 핸드 5장의 합으로 맞춰야 하며, 성공 시 강력한 보상(데미지 및 면역)을 얻습니다.
