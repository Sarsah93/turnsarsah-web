# 턴살사 (TurnSarsah) - 유저 매뉴얼

턴살사는 포커 족보를 활용한 1:1 턴제 카드 배틀 게임입니다. 플레이어는 카드를 선택하여 강력한 족보를 완성하고, 보스의 체력을 0으로 만들어 스테이지를 클리어해야 합니다.

---

## 1. 게임 기본 규칙
- **목표**: 총 10개 스테이지의 보스를 처치하여 최종 승리를 거둡니다.
- **체력(HP)**: 플레이어의 HP가 0이 되면 패배, 보스의 HP가 0이 되면 스테이지 클리어.
- **드로우 및 턴**: 
  - 매 턴 기본 2회의 카드 교체 기회가 주어집니다. 
  - 공격 후 사용한 카드는 자동으로 보충됩니다.

---

## 2. 공격 및 데미지 계산
족보 점수와 카드 숫자의 합산으로 데미지가 결정됩니다.

### 족보 보너스 (Hand Bonus)
| 족보 | 보너스 점수 |
| :--- | :--- |
| **Royal Flush** | 300 |
| **Straight Flush** | 175 |
| **Four of a Kind** | 150 |
| **Full House** | 125 |
| **Flush** | 100 |
| **Straight** | 75 (Wrap-around 포함) |
| **Three of a Kind** | 50 |
| **Two Pair** | 20 |
| **One Pair** | 10 |
| **High Card** | 0 (가장 높은 카드 2장의 합) |

- **조커(Joker)**: 모든 카드를 대체하며, 가장 높은 가치를 가집니다.
- **크리티컬**: 족보 구성에 Ace나 Joker가 포함될 경우, 장당 10%의 확률로 **1.25배** 데미지를 줍니다 (High Card 제외).
- **공격력 상한 (ATK CAP)**: 보스의 공격력은 모든 난이도에서 최대 **100**으로 고정됩니다.

## 3. 상태 이상 및 중첩 규칙
모든 디버프는 매 턴 종료 시 **15% 확률로 자동 해제**될 수 있습니다.

| 효과 | 설명 | 지속/중첩 |
| :--- | :--- | :--- |
| **출혈 (Bleeding)** | 매 턴 5 데미지 (30% 확률) | 4턴 |
| **과출혈** | 매 턴 15 고정 데미지 | 3턴 |
| **중독 (Poisoning)** | 턴마다 데미지 증가 (5, 10, 15, 20) | 4턴 |
| **쇠약 (Debilitating)**| 최대 체력 20% 감소 & 가하는 데미지 20% 감소 | 2턴 |
| **마비 (Paralyzing)** | 한 턴 동안 공격 불가능 (행동 불능) | 2턴 |
| **경감 (Reduction)** | 받는 데미지 X% 감소 (상태이상 데미지 제외) | 상시 (보스 전용) |
| **면역 (Immune)** | 적의 모든 디버프를 차단 | 3턴 |
| **재생 (Regenerating)** | 매 턴 체력 10 회복 (특정 조건 시 무한 갱신) | 3턴 |
| **회피 (Avoiding)** | 5% 확률로 적의 공격 무효화 | 상시 (기본) |

## 5. UI 가이드
- **공격**: 선택한 카드로 공격을 실행합니다.
- **교체**: 선택한 카드(최대 2장)를 덱의 새 카드로 바꿉니다. 한 스테이지 당 최대 2회.
- **일시정지 (ESC)**: 게임 저장, 설정 및 종료 메뉴를 호출합니다.
