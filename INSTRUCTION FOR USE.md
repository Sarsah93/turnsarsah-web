# 턴살사 (TurnSarsah) - 유저 매뉴얼

턴살사는 포커 족보를 활용한 1:1 턴제 카드 배틀 게임입니다. 플레이어는 카드를 선택하여 강력한 족보를 완성하고, 보스의 체력을 0으로 만들어 스테이지를 클리어해야 합니다.

---

## 1. 게임 기본 규칙
- **목표**: 각 챕터의 스테이지 보스들을 처치하여 최종 승리를 거둡니다.
- **체력(HP)**: 플레이어의 HP가 0이 되면 패배, 보스의 HP가 0이 되면 스테이지 클리어.
- **드로우 및 턴**: 
  - 매 턴 기본 2회의 카드 교체 기회가 주어집니다. 
  - 공격 후 사용한 카드는 자동으로 보충됩니다.

---

## 2. 공격 및 데미지 계산
족보 점수와 카드 숫자의 합산으로 데미지가 결정됩니다.

### 족보 보너스 (Hand Bonus)
| 족보 | 보너스 점수 |
| :--- | :--- |
| **Royal Flush** | 300 |
| **Straight Flush** | 175 |
| **Four of a Kind** | 150 |
| **Full House** | 125 |
| **Flush** | 100 |
| **Straight** | 75 (Wrap-around 포함) |
| **Three of a Kind** | 50 |
| **Two Pair** | 20 |
| **One Pair** | 10 |
| **High Card** | 0 (가장 높은 카드 2장의 합) |

- **조커(Joker)**: 모든 카드를 대체하며, 가장 높은 가치를 가집니다.
- **크리티컬**: 족보 구성에 특정 카드가 포함될 경우, 난이도별 확률로 데미지가 증폭됩니다.
- **공격력 상한 (ATK CAP)**: 보스의 공격력은 설정된 상한까지만 상승합니다.

---

## 3. 상태 이상 및 중첩 규칙
전투 중 발생하는 변수로, 다양한 효과가 부여될 수 있습니다.

| 효과 | 설명 | 지속/중첩 |
| :--- | :--- | :--- |
| **출혈 (Bleeding)** | 매 턴 고정 데미지 부여 | 4턴 |
| **과출혈** | 출혈이 심화된 상태로 보다 높은 고정 데미지 부여 | 3턴 |
| **중독 (Poisoning)** | 턴마다 데미지가 점진적으로 증가 | 4~6턴 |
| **쇠약 (Debilitating)**| 최대 체력 및 가하는 데미지 20% 감소 | 3턴 |
| **마비 (Paralyzing)** | 한 턴 동안 행동이 불가능해짐 | 2~3턴 |
| **경감 (Reduction)** | 받는 데미지를 일정 비율 감소시킴 | 상시 |
| **면역 (Immune)** | 모든 신규 디버프 부여를 차단 | 3턴 |
| **재생 (Regenerating)** | 매 턴 체력을 일정량 회복 | 3턴/상시 |
| **회피 (Avoiding)** | 일정 확률로 적의 공격을 완전히 무효화함 | 상시 |

---

## 4. UI 및 조작 가이드
전장 상황을 파악하고 최선의 전략을 실행하기 위한 인터페이스 안내입니다.

- **공격 (ATTACK)**: 현재 선택한 카드들로 족보를 구성하여 보스를 공격합니다.
- **교환 (SWAP)**: 패가 좋지 않을 때 선택한 카드를 덱의 새로운 카드로 바꿉니다. (스테이지당 횟수 제한)
- **상태이상 아이콘 (Condition Icons)**:
  - 플레이어 및 보스 HP바 근처에 현재 적용된 상태이상 아이콘이 표시됩니다.
  - **아이콘 클릭 시**: 해당 상태이상의 남은 턴수와 자세한 효과 설명을 담은 **툴팁 팝업**이 나타납니다.
- **일시정지 및 설정 (ESC)**: 게임을 저장하거나 사운드 설정, 메인 화면으로 나가기 등을 수행할 수 있습니다.
- **메시지 로그**: 화면 중앙의 텍스트를 통해 보스의 공격 성공 여부나 특수 기믹 발동 여부를 확인할 수 있습니다.

---

## 5. 챕터별 보스 데이터 매뉴얼
각 챕터마다 고유한 환경과 보스 군단이 플레이어를 기다립니다.

### 🌳 챕터 1: 들판 (Meadow field)
*   **테마**: 점점 강력해지는 고블린 무리들과의 전투입니다. 처음에는 평범한 고블린으로 시작하지만, 기병, 샤먼, 정예병을 거쳐 최종적으로는 고블린들의 군주와 마주하게 됩니다. 
*   **주요 규칙**: 카드 사용 금지(Rank/Suit Banned), 뒤집힌 카드(Blind), 족보 금지 등 기초적인 전술 제약이 적용됩니다.
*   **핵심 기믹**: 
    - **상태이상 공세**: 보스들은 단계별로 출혈(1~4), 중독(5), 마비(7) 등 다양한 상태이상을 부여하여 플레이어를 압박합니다.
    - **공격력 성장 및 기믹**: 후반부 정예/거대 보스는 공격이 적중할 때마다 공격력이 상승하거나(7), 2턴에 한 번 강력한 공격을 시도하는 등(8) 고유한 패턴을 가집니다.
    - **상시 경감**: 스테이지 7 이상의 보스들은 플레이어의 공격을 일정 비율로 항상 경감하여 받아냅니다.
    - **군주의 각성**: 최종 보스는 체력이 낮아지면 1회에 한해 모든 능력을 회복하고 각성합니다.

### 🏜️ 챕터 2A: 사막 (Desert)
*   **테마**: 뜨거운 태양과 모래 폭풍이 몰아치는 사막의 고대 괴수들과 언데드 군단입니다.
*   **진행 예고**: 미라와 모래뱀을 시작으로, 전설 속의 스핑크스와 같은 강력한 수호자들이 등장할 예정입니다.
*   **환경 영향**: 사막 특유의 환경적인 제약과 강력한 독성 공격이 주를 이룰 것으로 예상됩니다.

### 🌲 챕터 2B: 깊은 숲 (Deep forest)
*   **테마**: 빛이 들지 않는 숲속을 지배하는 오크 부족과의 치열한 생존 싸움입니다.
*   **진행 예고**: 일반 오크 전사부터 시작하여 오크 추장, 그리고 숲의 최상위 포식자인 하이 오크 로드까지 이어지는 계층적인 적들이 등장합니다.
*   **특징**: 고블린 무리보다 훨씬 탄탄한 체력과 압도적인 물리 공격력을 바탕으로 압박해올 것입니다.
